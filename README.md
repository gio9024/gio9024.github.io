# gio9024.github.io

Static GitHub Pages site for a personal project portfolio.

## Structure

- `index.html`: professional landing page for the site
- `assets/css/site.css`: shared styles
- `projects/index.html`: projects directory
- `projects/latinbiota/index.html`: Spanish landing page for the Latinbiota project
- `projects/latinbiota/report-es.html`: standalone interactive Latinbiota report

## Local preview

You can preview the site locally with a simple static server from the repo root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
